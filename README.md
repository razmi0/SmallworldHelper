# Smallworld Helper

### Features

- Add player
- Add starting score
- Delete player
- Reset player's score
- Add player's score
- Sort by score
- Save session
- Load session
- Line chart
- Bar chart
- Pie chart
- Undo / Redo actions
- Draggable players & cards
- Notifications
